### How to Install and run the Project

#### Installing the project
```
npm install
```

#### Running the project (Access http://localhost:5173/)
```
npm run dev
```

#### Running storybook (Access http://localhost:6006/)
```
npm run storybooy
```
While running storybook:
```
npm run test-storybook
```
